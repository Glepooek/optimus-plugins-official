## 快速开始

1. **先读方法论** — 理解 CQG 公式和三大卡点解法
2. **工具箱随身带** — 遇到问题时快速查阅
3. **做练习** — 每个练习至少做一遍，不熟练的重做

## 核心公式

```
好问题 = 背景(Context) + 问题(Question) + 目标(Goal)
```

## 三个卡点

| 卡点 | 解法 |
|---|---|
| 不知道问什么 | 5个启动器（是什么/为什么/怎么做/什么条件/如果…会怎样） |
| 问太模糊 | 精炼四步法（加限定词→加具体例子→加对比） |
| 不会追问 | 三条追问链（为什么/如果/怎么） |
