## [1.2.0] - 2026-06-30

### Added
- Step 0 参数解析：支持 `source=<路径>` 和 `target=<路径1,路径2,...>` 参数语法
- 支持 target 多目标（逗号分隔，解析为数组）
- 自定义路径边界处理：target 父目录不存在时 CHECKPOINT 询问是否创建，target 是文件时报错跳过
- 使用自定义参数时自动打印参数语法说明

### Changed
- 步骤编号整体 +1（原 Step 0-4 → 新 Step 1-5），新 Step 0 为参数解析

## [1.1.0] - 2026-06-30

### Added
- frontmatter `version` 字段（规范补充）

### Changed
- 符号链接目标验证：已存在的链接若目标路径不匹配，自动删除并重建（原来只跳过）
- 输出提示优化：跳过时显示"✔ 已是最新，跳过"，自动修正时显示"🔄 更新链接"

### Fixed
- Windows Step 2 `if` 块格式错误（多余空格）
- Bash 扫描源目录改用 `find -print0`，修复目录名含空格时数组解析错误

## [1.0.0] - 2026-06-28

### Added
- 初始版本：支持 Windows（符号链接权限预检）和 macOS/Linux 双平台
- 步骤：源目录扫描 → 目标目录同步 → 失效链接检测 → CHECKPOINT 清理
- 目标目录：`~/.claude/skills/` 和 `~/.kiro/skills/`（kiro 不存在则跳过）
- 反例黑名单（6 条）和全局使用说明
