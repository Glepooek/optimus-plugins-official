# 部门缩写对照表

## 缩写规则

子目录命名格式：`<部门缩写>_<项目名称>`

示例：`GX_统一认证平台`、`GJ_英语教材配套平台`

## 部门列表

| 序号 | 部门全称 | 缩写 | 子目录示例     |
|------|----------|------|-----------|
| 1 | 技术共享中心 | GX | `GX_项目名称` |
| 2 | 高等教育事业部 | GD | `GJ_项目名称` |
| 3 | 基础教育事业部 | JC | `JJ_项目名称` |
| 4 | 语言服务中心 | YF | `YF_项目名称` |
| 5 | 职教与中文事业部 | ZJ | `ZJ_项目名称` |
| 6 | 学历教育事业部 | XL | `XL_项目名称` |

## 使用说明

- 多部门协作项目：使用主责部门的缩写
- 跨部门共享文档：优先放入 `GX`（技术共享中心）目录
- 不确定归属：咨询项目负责人后再上传
