# 推荐技术栈

> 如果项目已有技术栈，以项目实际为准，本文件仅作为新建项目的默认选型。

| 层 | 技术 | 版本 | 理由 |
|----|------|------|------|
| 框架 | React | 18+ | 训练数据最丰富，AI 生成准确率最高 |
| 语言 | TypeScript | 5.x | strict 模式，AI 类型推断最成熟 |
| 样式 | Tailwind CSS | 3+ | 原子化 class，AI 还原设计稿零歧义 |
| 样式工具 | CVA + tailwind-merge + clsx | - | 组件变体 + class 合并 + 条件拼接 |
| 状态（客户端） | Zustand | - | 极简 API，AI 生成无模板代码 |
| 状态（服务端） | TanStack Query | v5 | 缓存/重试/失效 AI 一次写对 |
| 构建 | Vite | 5+ | 零配置 + 极速 HMR |
| 请求 | axios | 1.x | 拦截器 + 类型泛型 AI 最熟练 |
| UI 库 | Ant Design 5 / shadcn/ui | - | 组件覆盖全，AI 文档记忆最完整 |
| 路由 | React Router | v6 | AI 生成 loader/action 模式最稳 |
| 埋点 | 神策 SDK（sa-sdk-javascript） | - | 产品级数据分析，AI 熟悉其 API |
| 包管理器 | pnpm | - | - |
